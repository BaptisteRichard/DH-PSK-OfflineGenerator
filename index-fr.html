Instructions complètes

Cette page est un générateur de clé secrète partagée en deux phases utilisant l'agorithme de Diffie-Helman.
Elle n'utilise que du Javascript, vous pouvez donc la télécharger et l'utiliser localement pour plus de sécurité. (clic droit ici, enregistrer le lien sous...)
Les deux partenaires doivent utiliser cette page pour générer la clé partagée, donc envoyez un lien vers cette page ou le fichier sauvegardé &àgrave; votre partenaire.

Phase 1

  1. Saisissez une graine unique et aléatoire (ou utilisez celle pré-remplie) consitituée uniquement de caractères alphanumériques. Il est recommandé d'utiliser minimum 12 caractères, et vous en aurez &àgrave; nouveau besoin pour la phase 2, donc notez l&àgrave; ou ne fermez pas la fenêtre avant d'avoir terminé le processus
  2. Cliquez sur "Re-générer la phase 1"
  3. Copiez tous les chiffres affichés et envoyez les &àgrave; votre partenaire par le canal non sécurisé de votre choix (Mail, Signal, Google Talk, Messenger...)

Phase 2

  1. Collez les chiffres envoyés par votre partenaire (pas les votres) dans la boite prévue.
  2. Cliquez sur "Générer le mot de passe"
  3. (optionnel) Confirmez avec votre partenaire que vous avez le même code de validation.
  4. Copiez votre mot de passe partagé et voil&àgrave; !
